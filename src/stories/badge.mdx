# Badge Component

The `Badge` component is a versatile UI element used to highlight or categorize content. It is commonly used to display statuses, labels, or counts in a visually distinct manner.

## Shared Props

The `Badge` component supports a variety of props to customize its appearance and behavior. These props ensure flexibility and consistency across different use cases.

## Customization

The `Badge` component can be customized in various ways to suit your design needs:

- **Size**: Adjust the size of the badge using the `size` prop. Options include `sm`, `md`, and `lg`. The default size is `md`.
- **Color**: Define the color of the badge using the `color` prop. For example, `color="gray"`.
- **Label**: Set the text displayed inside the badge using the `label` prop.

These customization options allow you to create badges that fit seamlessly into your design system.

## Usage

```tsx
import { Badge } from 'azeriand-library';

<Badge size="md" color="gray" label="Success" />
```
