# Input Component

The `Input` component is a versatile and customizable text input field. It supports various configurations, including icons, max length, centered text, and disabled states.

## Customization

The `Input` component can be customized in various ways to suit your design needs:

- **Value** and **Placeholder**.
- **Max Length**: Limit the number of characters a user can input using the `maxLength` prop.
- **Icons**: Add icons to the input field using the `icon` and `iconPosition` props. Icons can be placed on the left or right side of the input.
- **Centered Text**: Center the text within the input field using the `centerText` prop.
- **Disabled State**: Disable the input field using the `disabled` prop, making it non-editable.

These customization options allow you to create a wide range of input styles while maintaining the library's cohesive design language.

## Usage

```tsx
import { Input } from 'azeriand-library';

<Input placeholder="Enter text here" maxLength={50} />
```
