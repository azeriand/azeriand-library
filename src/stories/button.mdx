# Button Component

The `Button` component is a versatile and customizable UI element. It supports various styles, sizes, and configurations to suit your design needs.

## Shared Props

The `Button` component inherits some props from the `Card` component, ensuring consistency and flexibility when customizing components.

## Customization

The `Button` component can be customized in various ways to suit your design needs:

- **Label**: Define the text label of the button using the `label` prop.
- **Appearance**: Define the appearance of the button using the `appearance` prop. Options include `ghost`, `mate`, and `outlined`.
- **Size**: Adjust the size of the button using the `size` prop. Options include `sm` for small buttons.
- **Icon**: Add an icon to the button using the `icon` prop. You can position the icon on the left or right using the `position` prop.
- **No Padding**: Enable or disable the default padding using the `noPadding` prop.
- **No Blur**: Enable or disable the glass-like blur effect using the `noBlur` prop.
- **Color**: Define the primary color of the button using the `color` prop. For example, `color="blue"`.
- **Intensity**: Adjust the intensity of the color using the `intensity` prop. Higher values create a more vivid appearance.
- **Custom Styles**: Apply custom styles using the `style` prop or Tailwind classnames.

These customization options allow you to create a wide range of styles, from subtle to bold, while maintaining the library's cohesive design language.

## Usage

```tsx
import { Button } from 'azeriand-library';

<Button label="Click Me" color="blue" intensity={500} onClick={() => alert('Button clicked!')} />
```
